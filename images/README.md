# PantryChef Assets & Custom Images Folder

This folder is created for storing your food and recipe images! 

## 📸 How to use this folder:
1. Save your custom images here (e.g. `apples.jpg`, `spaghetti.png`, or `my-dish.webp`).
2. In the application code (`app.js` or `index.html`), reference them relatively like `images/apples.jpg` to display them in your cards or background assets.
3. You can drop in any recipe photos you snap yourself, and future updates can fetch and render these local assets automatically based on ingredient match!

*Enjoy cooking!* 🍳
